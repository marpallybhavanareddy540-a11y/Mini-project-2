Escape Room Challenge

Overview

Escape Room Challenge is a simple console-based Java game that tests players' programming knowledge through a series of multiple-choice puzzles. Players begin with 3 lives and must solve 4 rooms by answering Java-related questions correctly. Every incorrect answer costs one life. The objective is to escape before all lives are lost.

Features

- 🎮 Console-based interactive gameplay
- ❤️ Three-life system
- 🧩 Four programming puzzles
- ✅ Input validation using exception handling
- 🚪 Progressive room unlocking
- 🏆 Win/Lose game outcome

How to Run

1. Install Java JDK (8 or later).
2. Save the source code as "Main.java".
3. Open a terminal in the project directory.
4. Compile the program:

javac Main.java

5. Run the program:

java Main

Gameplay

The game consists of four rooms:

Room 1

Question on arithmetic operator precedence.

Room 2

Question on Java object creation.

Room 3

Question on loop execution.

Room 4

Question on the OOP concept of polymorphism.

Players lose one life for every incorrect answer. The game ends when:

- All four questions are answered correctly (Victory 🎉)
- All three lives are lost (Game Over)
